Nonparametrics

When to Use

when do not have knowledge about underlying distribution

samples do not have to come from normal distribution


Key Points

Robustness: insensitive to slight deviations from model assumptions; medians are robust, better for outliers

Rank: a number assigned to a datapoint according to its order in a sorted list; use if there are classification variables and error bars

Breakdown Point: a measure of robustness; the minimal fraction of contamination before a test no longer works

Concordant: object who is higher on one variable is also higher on the other variable

Discordant: object who is higher on one variable is lower on the other variable


Tests

Sign Test:

assigns data points as + or - depending on if it is above or below the expected value respectively

then see if it is significant that there are more + or more -; data must be randomly selected

Does the median of the variables have median mo?

Ho: m = mo

H1: m ≠ mo


Wilcoxon Signed Ranks:

use to see if two samples have different medians by testing if the matched-pair differences have a median other than 0

find the differences in the paired samples and rank them

sum the ranks of the positive and negative differences

use the smaller sum for the W statistic

Ho: matched-pair differences with a median of 0

H1: matched-pair differences with a non-zero median


Wilcoxon Rank Sum:

use ranks of sample data from two independent populations to test if they have different medians

asign ranks to all of the data points

sum the ranks of each sample

z = (T - μ)/σ where μ = n1(n1+n2+1)/2 and σ2 = n1n2(n1+n2+1)/12

Ho: samples come from populations with equal medians

H1: samples come from populations with different medians


Kolmogorov-Smirnov:

used to compare the shape of distributions between two samples or between one sample and a known sample

maximum vertical distance between the two functions is measured

Ho: two samples are drawn from the same distribution

H1: two samples are drawn from different distributions


Cramer-von-Mises Statistic:

use to see if the cdf of the data and a model are different

describes global and local differences


Anderson-Darling Test

used if able to assess differences in outliers well by weighting the CvM statistic


Kendall-Tau's Correlation Test

used to determine if there is a relationship between two variables using concordant and discordant pairing

count the number of concordant and discordant pairs

z2 = 9 τ2(n(n-1))/2(2n+5) where τ = (C-D)/(C+D)


Spearman-Rho Test

used to determine if there is a relationship between two variables using the difference between ranks

give each object a ranking is each variable

find the difference between the ranks for each object

ρ = 1 - 6∑diff2/n(n2-1)
