Censored and Truncated Data

When to Use

when wanting to account for bias in missing data and to form better conclusions based on the missing data


Key Points

Truncated: occurs when the data are not observed beyond a certain threshold or limit

Censored: occurs when the exact time of an event of interest is not known for some observations

Upper-Limit (left-censored): maximum value to which an object may have but the value is too low to be detected

Survival Function: S(x) = P(X>x) = 1 - F(x) where F(x) is CDF

probability that an object has value above specific value x

Hazard Function: h(x) = f(x)/S(x) = -d/dx(lnS(x)) where f(x) is PDF;

what is the probability that a person will die at a certain age

Kaplan Meier Curve: used to graphically represent the survival rate at different times;


Techniques

Cox Regression:

proportional hazard

used to determine the effect of different variables on survival time

coefficients near 0 indicate no effect

'' greater than 0 indicate increased risk

'' lower than 0 indicate decreased risk


Gehan's Test:

used to see if 2 survival curves are significantly different

similar to the Wilcoxon test (see Nonparametrics)

Uij = +1 if xi1 < xj2 (where xi1 may be censored)

Uij = -1 if xi1 > xj2 (where xj2 may be censored)

Uij = 0 if xi1 = xj2 (or if relationship is ill determined)

WGehan = ∑Uij


Logrank Test:

used to see if 2 survival curves are significantly different

Ho: survival functions are identical

H1: survival functions are different

for each event times calculate the expected value for the group

use the covariance to calculate the test statistic


Kaplan-Meier Estimator:

used for getting the survival curve of censored data<

the effective censoring needs to be random


Lynden-Bell-Woodroofe Estimator:

used for getting the survival curve of truncated data

S(x) = P(X ≤ x| X ≤ umin) S(umin)

can only get survival curve of a range where ui is the sensitivity limit
