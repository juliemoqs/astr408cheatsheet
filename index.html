Welcome to Julie's Cheat Sheet for Statistical Methods in Physics and Astronomy

This website was created as a place I can go back to whenever I need some information about statistical methods that may be useful depending on the type of data I am working with. This was created for the my ASTR408 class at Rice University. Each topic will include a description of when to use these techniques, keypoints, examples where needed, and a funny (typically cat) meme.



For more references and/or detail, I suggest Modern Statistical Methods for Astronomy (with R applications) by E. Feigelson and G. Babu.
